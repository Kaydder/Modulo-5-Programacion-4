# Biblioteca Personal con KeyDB y redis-py

## Objetivo
Modificar la aplicación de biblioteca personal para utilizar KeyDB como sistema de almacenamiento en memoria, reemplazando las bases de datos relacionales o documentales por una arquitectura clave-valor con objetos JSON.

---

## Funcionalidades

- Agregar nuevo libro  
- Actualizar información de un libro  
- Eliminar libro existente  
- Ver listado de libros  
- Buscar por título, autor o género  
- Salir del programa  

---

## Requisitos

- Python 3.8 o superior  
- KeyDB instalado y en ejecución  
- Dependencias listadas en `requirements.txt`

Instalar dependencias:

```bash
pip install -r requirements.txt
